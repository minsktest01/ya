схема messenger (messenger://)

  • открыть список чатов
    • link - messenger://chat/list
  • открыть чат БЕЗ возможности присоединиться
    • link - messenger://chat/open?chat_id=0/0/b3589d8f-a435-4c82-9edb-cb8de8ab610c
  • открыть чат с возможностью присоединения
    • link - messenger://chat/invite?chat_id=0/0/5ce38cc1-f534-4393-8343-b3cb6a103dd8
  • открыть последний чат с последним непрочитанным сообщением
    • link - messenger://chat/open/last_unread
  • открыть настройки
    • link - messenger://settings
  • открыть персональный чат
    • link - messenger://user?user_id=4acde406-edda-4989-96e9-ed6142e57ae3
